# Kelly Criterion 在预测市场的应用

**来源：** https://arxiv.org/html/2412.14144v1

**核心 Takeaway（一句话）：**
预测市场价格不等于概率——Kel ly公式给出理性投资人该投的比例，但真实市场存在摩擦，导致价格与概率之间出现系统性缺口。

---

**详细内容：**

- [事实] 预测市场（如Polymarket）的合约价格 ≠ 事件概率。风险偏好、 payouts 不对称、杠杆效应都会使价格偏离。
- [事实] Kelly fraction 最优解：f* = (Q - P) / (1 + Q)，其中 P = p/(1-p)，Q = q/(1-q)，p = 市场价，q = 主观信念。
- [事实] 当 q = p（信念等于价格），f* = 0，不下注；当 q > p（认为赔率划算），Kelly 说正仓；当 q < p，Kelly 说负仓（做空）。
- [事实] 简化双投资者模型中，一个确信者 + 一个普通信念者，最大价格-信念缺口可达到显著规模。
- [推论] 这解释了为什么 Polymarket 上即便 event probability 接近 50%，价格可能显示 60% 或 40%。
- [事实] 文章提出改进 payout 结构以缩小缺口，并分析了biased coin模型中错误估计bias或错误计算fraction对增长率的影响。
