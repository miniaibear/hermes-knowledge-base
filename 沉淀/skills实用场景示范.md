# 沉淀：Skills 实用场景示范

## 触发场景
用户问："skills 有什么用？" → 这是 Query 流程的自然入口

## 回答摘要

Skills 是**技能的标准化封装**，三个核心价值：

1. **工作流自动化** — 把"第一次做对"的经验固化成可复用的步骤，以后同类任务一键执行，不用每次重新摸索
2. **跨 session 记忆** — 笔记有记忆上限，skills 把重要的工作流持久化，下次遇到同类问题直接加载
3. **上下文压缩** — 把长篇操作规范压缩成触发词 + 步骤清单，大幅降低每次重新学习的成本

## 什么时候用 Skills

| 场景 | 不用 Skills | 用 Skills |
|------|------------|-----------|
| 第一次做某件事 | 边想边做，可能踩坑 | 加载 skill，按步骤执行 |
| 第二次做同类事 | 翻聊天记录 | `/skill-name` 直接加载 |
| 需要分享给其他 AI | 复制粘贴一堆上下文 | 告诉他 skill 名字就行 |

## 关键洞察
> Skills 是"搜索的逆过程" — 搜索是找已存在的东西，Skills 是把已做过的事结构化存档以便复用。
