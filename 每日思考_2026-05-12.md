### 核心诊断

**五件事不是独立的，是一条链：**

```
4月20日卖飞5万 → 损失财务预期 → 
队友隐瞒 → 信任受损+二次伤害 → 
空投不到账+网贷 → 财务焦虑慢性化 →
Web3拖半年 → 自我能力怀疑累积 →
回忆丢失 → 身份根基动摇 →
= 系统性崩盘
```

**自卑的怪圈：**
- 本金到位但卖飞了 → "我果然不行"
- 更不敢做大的决定 → 机会来了又拿不住 → "我果然不行"
- 回避一切有风险的动作 → Web3学半年不动 → "我果然什么都做不成"
- 执行力彻底冻结

**队友隐瞒的核心伤害：**
你怀疑他故意隐瞒。这个怀疑一旦成立，你受到的不是2万块的伤害，而是"这个世界上没有人真的希望你赢"的伤害。这种感觉会让一个人把所有社交关系都看成风险——然后更孤立，更难行动。

### 真正的问题是什么

**你的执行力丧失不是懒，是受伤。**

- 每一个行动都连着一个巨大的心理包袱（那5万）
- 每一次尝试都觉得自己在证明"我是不是真的不行"
- 长期财务焦虑让你的大脑始终处于警戒状态
- 身份丢失让你做任何事都没有根基

**这不是拖延。这是心理资源耗尽后的停机保护。**

### 阿德勒的解法

1. **找一个具体的小事，先做成一件**
   - 不是Web3，不是炒币，是某个5分钟能完成的事
   - 从行动结果中重新建立"我是有用的"的证据

2. **停止重演那5万**
   - 复盘写完就够了，不要反复在脑子里播放
   - 反复重演不会让你下次做得更好，只会让神经回路更习惯"失败感"

3. **和队友摊牌谈一次**
   - 把怀疑说出来，消除不确定性
   - 不确定比坏消息更消耗人

4. **关于身份丢失**
   - 这不是学习能解决的，是行动的结果
   - 你在做有价值的事情的过程中，才能重新建立"我是谁"
